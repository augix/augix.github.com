<python> #!/usr/bin/env python from math import sqrt def quadratic_roots(a,b,c):

discriminant = b*b-4*a*c if discriminant >= 0: x0 = (-b + sqrt(discriminant))/2.0/a x1 = (-b - sqrt(discriminant))/2.0/a return (x0, x1) else: return (None, None)

Exercise 5: factorial

<python> #!/usr/bin/env python def factorial(n):

assert (n>0), 'n is not positive.' nu = range(n,0,-1) res = 1 for i in nu: res = res * i return res

print factorial(1) print factorial(2) print factorial(3) print factorial(4) print factorial(5) print factorial(6) </python>
Result: <pre> >>> factorial(2) 2 >>> factorial(3) 6 >>> factorial(4) 24 >>> factorial(5) 120 >>> factorial(6) 720 >>> factorial(1) 1 </pre>
