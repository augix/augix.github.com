split

split a string into a vector: <R> x = "string_x" strsplit(x, split="") </R> <pre> 1 [1] "s" "t" "r" "i" "n" "g" "_" "x" </pre>

paste

<R> x = paste("Hello", "World", sep=", ") print(x) </R> <pre> [1] "Hello, World" </pre>

<R> chars = c("m", "e", "r", "g", "e", "d") paste(chars, collapse = "") </R>

<pre> [1] "merged" </pre>

toString

<R> x = c("Hello", "World") y = toString(x) print(y) </R> <pre> [1] "Hello, World" </pre>

ps: it seems "format" is also an useful function.

chartr

<R> x = "ACGTGTGACGT" y = chartr("acgtACGT","tgcaTGCA",x) print(x);print(y) </R>

<pre> [1] "ACGTGTGACGT" [1] "TGCACACTGCA" </pre>

<R> DNA.Rev.Comp <- function(x) { if (is.character(x) && length(x) == 1) { x = chartr("acgtACGT","tgcaTGCA",x) x = unlist(strsplit(x,split="")) x = rev(x) x = paste(x,collapse="") return(x) } else { warning("Wrong argument type in DNA.Rev.Comp, NA returned") return(NA) } }

DNA.Rev.Comp("ACTGTGACCACGTCGTA") </R>
