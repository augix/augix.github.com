  • Double dictionary

reference: create a double dictionary in python

the following WON'T work <python> >>> x = dict() >>> x[1][1] = “foo” </python> instead, one can use this: <python> >>> x = dict() >>> x[1,1] = “foo” </python>

note that, tuple (1,1) is used as key.
