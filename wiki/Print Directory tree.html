$ ls -R | grep ":$" | sed -e 's/:$//' -e 's/[-][\/]*\//--/g' -e 's/^/ /' -e 's/-/|/'

<pre> |-Application Support |---AddressBook |-----Images |---Adium 2.0 |-----Contact List |-----Emoticons |-----Message Styles |-----PlugIns |-----Scripts </pre>

One may also save this command line as ~/bin/tree then one can just type $ tree ./

What I did is making an alias: <pre> $ vi ~/.bashrc alias tree="ls -R | grep ":$" | sed -e 's/:$//' -e 's/[-][\/]*\//--/g' -e 's/^/ /' -e 's/-/|/'" $ tree </pre>

only display the folders with depth < 3
