Jumping

One can use CTRL-O or CTRL-I jumping to older cursor position or newer position. One can also use 

:ju

to print the jump list.

Buffers

  • switch to previous buffer: <c-6>

Open file

gf	 open in the same window ("goto file")
<c-w>f	 open in a new window (Ctrl-w f)
<c-w>gf	 open in a new tab (Ctrl-w gf)

Files and Tabs

  • create a new tab 
    :tabnew
  • switch to next tab 
    gt

or 

:tabn

Window

Move cursor to other windows. 

CTRL-W CTRL-W                       Move to the next window
CTRL-W j,k,h,l                      Move down, up, lfet, right
CTRL-W <Down>,<Up>,<Left>,<Right>   Move down, up, lfet, right
