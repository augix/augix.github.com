Start position of a match

Category:Perl If you want to find the position of a match in an incoming string, simply check the length of '''$`''' (That's $PREMATCH if you've chosen to use English;) to check where it starts, and add the length of '''$&''' (that's $MATCH) to find where it ends.

Let's say I want to find all the URLs refered to in a web page that's loaded into the variable $html. I could write:

<perl> push @section,[length($`),length($&),$1]

while ($html =~ m/(https?://[^ >"]+)/g);

</perl>

and that will give me a list of 3-element lists containing start point, length and actual string matched. Here's the code to display that list:

<perl> foreach $element(@section) {

print (join(", ",@$element),"\n");

} </perl>

P.S. I loaded my whole web page into a single variable using the code

<perl> open (FH,"/Library/WebServer/live_html/resources/index.html"); undef $/; $html = <FH>; </perl>

So, the whole script could be: <perl> open FH,$ARGV[0] or die $!; undef $/; $html = <FH>;

push @section,[length($`),length($&),$1]

while ($html =~ m!(https?://[^ >"]+)!g);

foreach $element(@section) {

print (join(", ",@$element),"\n");

} </perl>
