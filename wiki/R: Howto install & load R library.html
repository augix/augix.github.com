R: Howto install %title% load R library

Category: R <bash> $ mkdir ~/Library/R/ $ cd ~/Library/R/

$ R CMD INSTALL -l ~/Library/R/ seqinr_1.0-6.tar.gz > library(seqinr, lib.loc="~/Library/R/")

  1. or you can

$ R CMD INSTALL -l seqinr seqinr_1.0-6.tar.gz > library(seqinr, lib.loc="~/Library/R/seqinr")

</bash>

Don't have to specify '''lib.loc''' if the variable '''$R_LIBS''' is already set in the '''~/.bashrc''' like this: <bash> R_LIBS=~/Library/R/ export R_LIBS </bash>
